# Native Custom Fields User Guide

Welcome to the official documentation for *Native Custom Fields*. This guide is designed to help you get started with the plugin, from installation to creating complex data structures using WordPress's native interface components.

## Chapters

### 1) Installation Guide
Step-by-step instructions on how to download, install, and activate the plugin on your WordPress site.

### 2) Creating Post Types
How to create and manage custom content types without writing code.

### 3) Adding Meta Fields
Learn how to create field groups and add custom fields using native WordPress UI components.

### 4) Creating Taxonomies
Learn how to organize your content using custom categories and tags.

### 5) Adding Fields to Taxonomies
Extend your categories and tags by adding custom metadata fields.

### 6) User Meta Fields
Learn how to add custom fields to user profiles for richer user information.

### 7) Options Pages
Learn how to create options pages for your plugin.
